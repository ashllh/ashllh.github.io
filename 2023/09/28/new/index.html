利用HEXO建立个人博客

前言

如你所见，这是我的第一篇文章。有一天我萌生了建立网站的想法，但是后来发现建立网站的成本比较高，于是最终决定建立我的个人博客，而且还是零成本的。我认为这件事实在是太酷啦，于是也想与你们分享建立个人博客的经验。

正文

1 . 注册GitHub账号(已经有账号的跳过这一步)

访问 GitHub (有可能进不去，可以用Watt Toolkit 加速) 点击右上角的 sign up 依次输入电子邮箱，设置密码和输入验证码完成注册(没有电子邮箱的可以注册163邮箱 或者QQ邮箱 )

2 . 下载Node.js和Git

下载Nodej.s

访问Node.js 网站，随便选一个版本下载

下载完后双击运行，除了下面这个勾选上其他都直接下一步，最后完成即可

下载Git

下载Git 根据自己电脑系统选择版本

下载完双击打开，一路下一步完成安装。

检验

完成上面的步骤后同时按Win键和R键输入”cmd”并回车，依次输入以下内容并回车，出现版本号即表明安装成功

node -v npm -v git --version

3 . 连接GitHub并建立仓库

创建SSH密钥

在桌面右键并点击”Open Git Bash here”打开窗口

依次输入以下内容，（用户名和邮箱即为注册GitHub时的用户名和邮箱）

git config --global user.name "用户名"

git config --global user.email "邮箱"

再创建SSH密钥

ssh-keygen -t rsa -C "GitHub 邮箱"回车三次

输入cat ~/.ssh/id_rsa.pub并复制其返还的所有内容（即公钥）（右键Copy来复制)

连接GitHub

进入GitHub，点击右边头像，点击右下角Settings

再点击SSH and GPG keys

点击New SSH Key，标题随便输入，key把之前复制的公钥粘贴进去

添加SSH Key

创建仓库

还是上面一张图的界面，右上角加号 >>New repository

Repository name 中输入”用户名.github.io“（必须是 你的用户名.github.io)

勾选 “Add a README file” 描述选填，点击最下面绿色按钮创建仓库

4 . 本地安装 Hexo 博客程序

创建博客目录

新建一个文件夹用来放置博客文件（建议在C盘以外的盘创建，名称最好是英文和数字)

打开这个文件夹，右键>>Open Git Bash here,输入 npm install -g hexo-cli 并回车，等待安装完成

Hexo 初始化和本地预览

依次输入并回车

hexo init # 初始化

npm install # 安装组件

hexo g # 生成页面

hexo s # 启动预览

这时候去浏览器访问 http://localhost:4000(把个链接输入到浏览器最上面的框，而不是下面的搜索框) 看到以下页面，那么本地博客安装成功！再回到命令窗口，按Ctrl +C结束

5 . 部署 Hexo 到 GitHub Pages

安装 hexo-deployer-git

还是之前的命令窗口，输入 npm install hexo-deployer-git --save 并回车

修改配置文件

还是之前的文件夹，双击打开 _config.yml 这个文件（我打开用的软件是VsCode ；想下载的看一下这个文章 )

将最后面的内容改成这样（冒号后面有1个空格，不能多也不能少），用户名就是你注册GitHub时的用户名

这是我的（绿色字部分不用管）

修改完之后一定要Ctrl + S保存

部署

输入 hexo g 回车，再输入 hexo d 回车 如果没有被标红的Fatal,那大概率就是成了。

6 . 成功创建博客

最后在浏览器地址栏输入 https://用户名.github.io 就可以看见专属于你的博客内容了！

尾声

写了怎么长一篇文章，我已经很累了，相信你也耗费了大量精力来完成以上工作。如果你有任何关于这篇文章的疑问或者按照上面方法出现的问题，你都可以来咨询我；如果你想要让你的博客个性化或者发布文章，可以去看看我的下一篇文章。
