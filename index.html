利用HEXO建立个人博客

利用HEXO建立个人博客

前言

如你所见，这是我的第一篇文章。有一天我萌生了建立网站的想法，但是后来发现建立网站的成本比较高，于是最终决定建立我的个人博客，而且还是零成本的。我认为这件事实在是太酷啦，于是也想与你们分享建立个人博客的经验。

正文

1 . 注册GitHub账号(已经有账号的跳过这一步)

访问 GitHub (有可能进不去，可以用Watt Toolkit 加速) 点击右上角的 sign up 依次输入电子邮箱，设置密码和输入验证码完成注册(没有电子邮箱的可以注册163邮箱 或者QQ邮箱 )

2 . 下载Node.js和Git

下载Nodej.s

访问Node.js 网站，随便选一个版本下载

下载完后双击运行，除了下面这个勾选上其他都直接下一步，最后完成即可

下载Git

下载Git 根据自己电脑系统选择版本

下载完双击打开，一路下一步完成安装。

检验

完成上面的步骤后同时按Win键和R键输入”cmd”并回车，依次输入以下内容并回车，出现版本号即表明安装成功

node -v npm -v git --version

3 . 连接GitHub并建立仓库

创建SSH密钥

在桌面右键并点击”Open Git Bash here”打开窗口

依次输入以下内容，（用户名和邮箱即为注册GitHub时的用户名和邮箱）

git config --global user.name "用户名"

git config --global user.email "邮箱"

再创建SSH密钥

ssh-keygen -t rsa -C "GitHub 邮箱"回车三次

输入cat ~/.ssh/id_rsa.pub并复制其返还的所有内容

连接GitHub
